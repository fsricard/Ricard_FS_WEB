## 😈 ¿Qué demonios es esto?

El **Diablillo Sarcástico** es un proyecto web donde un pequeño ente infernal:

- Te responde con sarcasmo  
- Te vacila con cariño  
- Te acompaña mientras trabajas  
- Y te recuerda que la vida es demasiado corta para interfaces aburridas  

Vamos, lo que viene siendo un *asistente digital con actitud*.

---

## 🔥 Características principales

- Respuestas sarcásticas generadas dinámicamente  
- Personalidad configurable (del “troll suave” al “diablillo profesional”)  
- Interfaz cartoon‑realista con animaciones  
- Backend en PHP con lógica modular  
- Frontend ligero con JS y CSS  
- Sistema de frases inteligentes  
- Estructura escalable para futuras maldades

## 😈 Cómo invocar al Diablillo

Comando para iniciar servidor local:  
php -S localhost:666

Abrir en el navegador:  
http://localhost:666

Si escuchas risitas malévolas… funciona.
