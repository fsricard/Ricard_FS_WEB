## 😈 Cómo invocar al Diablillo

Comando para iniciar servidor local:  
php -S localhost:666

Abrir en el navegador:  
http://localhost:666

Si escuchas risitas malévolas… funciona.

## 🔥 Instalación avanzada (para almas valientes)

1. Clonar el repositorio desde el inframundo:  
   git clone https://github.com/fsricard/Ricard_FS_WEB.git

2. Entrar en el círculo infernal del proyecto:  
   cd Ricard_FS_WEB

3. Configurar tu entorno:
   - PHP 8 o superior
   - Extensiones básicas activadas
   - Servidor local o Apache/Nginx
   - Permisos adecuados en la carpeta assets/img si quieres subir imágenes

4. Crear tu archivo de configuración (si procede):  
   Copiar config/database.php.example a config/database.php  
   Ajustar credenciales según tu servidor

5. Encender el fuego del servidor:  
   php -S localhost:666

6. Abrir el navegador y dejar que el diablillo haga el resto.

## 🔥 Errores comunes y cómo sobrevivir a ellos

**Error: El servidor no arranca.**  
Solución: Asegúrate de no haber invocado accidentalmente un demonio mayor. O revisa el puerto, lo que sea más rápido.

**Error: Página en blanco.**  
Solución: Activa display_errors o sacrifica un bug menor. Ambos funcionan.

**Error: Las rutas no cargan.**  
Solución: Revisa el .htaccess o culpa a alguien más. Tradición del gremio.

**Error: El diablillo no responde.**  
Solución: Dale café. O revisa la consola, pero el café suele funcionar.

**Error: El CSS no se aplica.**  
Solución: Limpia la caché. Sí, otra vez. Y otra. Y otra.

**Error: El proyecto funciona en local pero no en producción.**  
Solución: Bienvenido al infierno real. Revisa permisos, rutas absolutas y tu paciencia.
